# Todo App - Task Management Application

A simple, efficient, and user-friendly to-do list application with **local storage** functionality. Organize your tasks with priorities, due dates, and tags.

## ✨ Features

### Core Functionality
- ✅ **Add Tasks** - Create new tasks with title, priority, and due date
- ✏️ **Edit Tasks** - Modify task details anytime
- 🗑️ **Delete Tasks** - Remove completed or unwanted tasks
- ☑️ **Mark Complete** - Check off completed tasks
- 🔄 **Local Storage** - All tasks are automatically saved to browser storage
- 🔍 **Search & Filter** - Find tasks by name or tags
- 🏷️ **Tags** - Organize tasks with custom tags
- 📅 **Due Dates** - Set and track task deadlines
- ⚡ **Priority Levels** - Mark tasks as Low, Medium, or High priority
- 📊 **Statistics** - View task completion stats at a glance

## 🎯 Quick Start

```bash
cd todo-app
npm install
npm start
```

## 💾 Local Storage Features

✅ All tasks automatically saved to browser storage  
✅ Persists data even after closing the browser  
✅ No account or login required  
✅ Privacy-focused - data stays on your device  

## 📊 Statistics Dashboard

- **Total Tasks** - All tasks
- **Completed** - Marked as done
- **Active** - Not yet completed
- **High Priority** - Urgent tasks pending
